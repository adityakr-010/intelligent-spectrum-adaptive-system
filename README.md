# Intelligent Spectrum Detection & Adaptive Communication System
This project implements a real-time AI-assisted spectrum monitoring and adaptive communication system. 
It simulates dynamic RF environments, extracts spectral features, classifies channel conditions, and 
autonomously selects the optimal communication channel.

The system is inspired by cognitive radio and intelligent wireless communication systems used in UAVs, 
defence electronics, and IoT networks.

Key Features:
- Real-time RF signal simulation across multiple channels
- FFT-based spectral feature extraction
- Machine learning-based channel classification
- Channel quality scoring (0–100 scale)
- Autonomous decision engine for adaptive communication
- Live visualization dashboard with decision logs
